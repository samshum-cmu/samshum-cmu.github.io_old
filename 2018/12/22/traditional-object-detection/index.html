Object Detection by Traditional Algorithms

Stop Sign Detection using Traditional Computer vision algorithms


Source footage

The source video is from youtube, after performing some basic image processing technique using OpenCV,
The video is cropped and trimmed down to fit my usage.

First, lets take a look what happens if we simply use the red color as the “condition” to search for the stop sign.

Color

However, in my past experience dealing with color-based conditions… it tells me that,
the performance of such approach varies greatly in different scenarios..

For instance, with different ligh intensity and mutiple objects of similar color, the weakness of such approach is exposed.
Thus, only by using color is not that robust and stable…



Time to use some proper algorthims!
These algorthims are used to extract features,
by comparing the features of the template sample image and the input image,
we can perform feature-matching to search for the object. (At least thats how i do it)

Haar Cascade (HAAR)

Scale-Invariant Feature Transform (SIFT)

In my opinion, i think SIFT is probably one of the best, in terms of the accuracy and robustness.
Althrough it is slower than approaches such as ORB or SURF.
